Adicione este launch.json na pasta ./vscode/launch.json

```json
{
    "version": "0.2.0",
    "configurations": [
      {
        "name": "dev",
        "cwd": ".",
        "request": "launch",
        "type": "dart",
        "flutterMode": "debug",
        "program": "lib/main/main_dev.dart",
      },
     
    ]
  }
```
